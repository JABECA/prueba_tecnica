## Notas al proyecto

Este proyecto cuenta los módulos solicitados (usuarios, carros, ciudades y viajes) cada uno con su CRUD, también tiene una API que realiza la misma función que la API solicitada con SLIM, es decir tiene los tres endpoints del punto 2 del pdf. 

En la raíz del proyecto hay una carpeta que se llama postman donde encontrarán las colecciones para POSTMAN para cada punto (2-1, 2-2, 2-3) y también la colección conjunta de los tres endpoints para el API realizada en laravel (2. Coleccion postman API laravel.json) y otra coleccion conjunta para el API realizada con SLIM (2. Coleccion postman API SLIM.json).

También encontrarán una carpeta llamada consultas donde coloque las 6 consultas solicitadas en el punto 3 del pdf, sin embargo el el aplicativo en la opción informes cree enlaces para ejecutar las mismas 6 consultas y pintando los datos que traen dichas consultas.

El la raiz del proyecto encontraran un archivo llamado solucion_pregunta_4 el cual tiene la solucion de la pregunta 4 del pdf
